Legend - a System Reference Document distribution

A System Reference Document distribution

You may have heard of a certain Role Playing Game Company (let's call them Sorcerers of the Seaside or "SotS") that publishes their main gaming system in open source format called the "System Reference Document" (SRD). Well, Legend is a new distribution for the SRD. Legend repackages the SRD files into a more useful XML format and displays them for browsing on this web site.

Legal and License notes

Background

Wizards of the Coast has structured their licenses for the SRD in an interesting way. The SRD is covered under the Open Gaming License. The OGL does not license use of the names "d20" or "Dungeons and Dragons" or any other of the game names.

The System Reference Document is derived from the d20 system rules, but it must be clear that SRD is NOT d20. But on the other hand, you can assume that they are functionally the same, because that is the promise that Wizards has made and has kept so far.

Now the name "System Reference Document" is not particularly catchy (which is probably intentional). Thus this NEW distribution of the SRD has a new name which is hopefully a little more catchy than the original.
