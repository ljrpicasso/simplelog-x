Legend Downloads

Contributing

If you wish to contribute to Legend we welcome your contributions. It should be fairly simple. For the moment, please send in corrections to the root SRD material only -- we aren't set up to handle independent material yet, but that's something that I want to do later on.

Here's how to make it as easy as possible for me to accept your changes.

  • Make your modifications directly to the XML file
  • Make as few changes as possible (in other words, don't change line spacing, indentation, schema, XML format)
  • You need a tool called "diff" that creates "patch files"
  • Create a patchfile that contains the differences between the latest published version of Legend and your modified version
  • Please use the unified format (-ub options are good)
  • Send me the patchfile only (by email)

It's a bit of work but if you do that, it will make it very easy for me to integrate your changes into Legend.
